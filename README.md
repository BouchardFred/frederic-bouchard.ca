# frederic-bouchard.ca

Personal academic website for Frédéric Bouchard.

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

This site is configured for GitHub Pages using the workflow in `.github/workflows/deploy.yml`.

The custom domain is configured with the `CNAME` file:

```txt
frederic-bouchard.ca
```
